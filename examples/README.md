# Examples

This directory holds sanitized example configurations and reference output that complements the main configs in this repository.

Examples are not authoritative — they are illustrative. Use them as a starting point for understanding what a working configuration looks like, not as a drop-in replacement for the main config files in `configs/`.

## Layout

```
examples/
  audit-output-sample.txt   — Sample output from wp-permission-audit.sh
```

## Notes

- All examples use the documentation reserved range `203.0.113.0/24` for IP addresses
- All examples use `example.com` for domain names
- All credentials, keys, and account identifiers are clearly marked placeholders
- Example output reflects a hypothetical environment, not any real production system
